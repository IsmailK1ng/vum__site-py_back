Политика конфиденциальности

Мобильное приложение VUM ERP  ·  Последнее обновление: июнь 2026

Общие положения

Настоящая Политика конфиденциальности описывает, как мобильное приложение VUM ERP, разработанное компанией VAN UNIVERSAL MOTORS, собирает, использует и защищает данные пользователей.

Приложение предназначено исключительно для сотрудников компании VAN UNIVERSAL MOTORS. Доступ к приложению предоставляется только авторизованным пользователям.

Какие данные мы собираем

  • Учётные данные Логин и пароль сотрудника для входа в систему ERP. Пароли хранятся в зашифрованном виде.
  • Геолокация Местоположение устройства используется для производственных и логистических задач в рамках работы с ERP системой.
  • Камера (QR-сканер) Доступ к камере используется исключительно для сканирования QR-кодов в рамках рабочих процессов.
  • Фотографии Приложение может запрашивать доступ к галерее для загрузки фотографий в рамках рабочих задач.

Цели использования данных

Все собранные данные используются исключительно в следующих целях:

  • Аутентификация и авторизация сотрудников в системе ERP
  • Выполнение производственных и логистических задач компании
  • Обеспечение функциональности приложения
  • Отправка push-уведомлений о рабочих событиях

Защита данных

Мы применяем технические и организационные меры для защиты ваших данных:

  • Передача данных осуществляется по защищённому протоколу HTTPS
  • Данные хранятся на защищённых серверах компании
  • Доступ к данным ограничен только авторизованными сотрудниками
  • Данные не передаются третьим лицам

Права пользователей

Как сотрудник компании, вы имеете право:

  • Запросить информацию о хранящихся данных
  • Обратиться с запросом на изменение или удаление данных
  • Отказаться от сбора геолокации через настройки устройства

Privacy Policy

VUM ERP Mobile Application  ·  Last updated: June 2026

General Provisions

This Privacy Policy describes how the VUM ERP mobile application, developed by VAN UNIVERSAL MOTORS, collects, uses, and protects user data.

The application is intended exclusively for employees of VAN UNIVERSAL MOTORS. Access is granted only to authorized users.

Data We Collect

  • Account Credentials Employee login and password for accessing the ERP system. Passwords are stored in encrypted form.
  • Geolocation Device location is used for production and logistics tasks within the ERP system workflow.
  • Camera (QR Scanner) Camera access is used exclusively for scanning QR codes as part of work processes.
  • Photos The application may request access to the gallery for uploading photos as part of work tasks.

Purpose of Data Use

  • Authentication and authorization of employees in the ERP system
  • Execution of company production and logistics tasks
  • Ensuring application functionality
  • Sending push notifications about work events

Data Protection

  • Data is transmitted via secure HTTPS protocol
  • Data is stored on the company's secured servers
  • Data access is restricted to authorized employees only
  • Data is not shared with third parties

User Rights

  • Request information about stored data
  • Submit a request to modify or delete data
  • Opt out of geolocation collection via device settings
