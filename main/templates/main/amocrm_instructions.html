Шаг 1: Получение Authorization Code

1. Откройте ссылку для авторизации

Перейдите по следующей ссылке (замените YOUR_SUBDOMAIN на ваш поддомен):

https://www.amocrm.ru/oauth?client_id=YOUR_CLIENT_ID&mode=post_message

2. Авторизуйтесь в amoCRM

Войдите в систему под учетной записью с правами администратора.

3. Скопируйте Authorization Code

После авторизации откроется окно с кодом. Скопируйте значение code.

Важно: Authorization Code действителен только 20 минут. Используйте его сразу после получения.
Шаг 2: Получение токенов через API

1. Выполните POST-запрос

Используйте следующий curl-запрос (замените YOUR_SUBDOMAIN, CLIENT_ID, CLIENT_SECRET, CODE, REDIRECT_URI):

curl -X POST https://YOUR_SUBDOMAIN.amocrm.ru/oauth2/access_token \ -H "Content-Type: application/json" \ -d '{ "client_id": "YOUR_CLIENT_ID", "client_secret": "YOUR_CLIENT_SECRET", "grant_type": "authorization_code", "code": "YOUR_CODE", "redirect_uri": "YOUR_REDIRECT_URI" }'

2. Сохраните полученные токены

В ответе вы получите:

{ "access_token": "...", "refresh_token": "...", "expires_in": 86400 }
Шаг 3: Обновление токенов в админке

1. Откройте Django shell

python manage.py shell

2. Обновите токены

from main.models import AmoCRMToken from django.utils import timezone from datetime import timedelta token = AmoCRMToken.get_instance() token.access_token = 'YOUR_ACCESS_TOKEN' token.refresh_token = 'YOUR_REFRESH_TOKEN' token.expires_at = timezone.now() + timedelta(seconds=86400) token.save() print('Токены успешно обновлены')
Совет: После обновления токенов система будет автоматически продлевать их каждый час.
Автоматическое обновление токенов

Токены автоматически обновляются через:

  • Кнопку "Обновить токен" в админке
  • Автоматически при отправке заявок, если токен истёк
Внимание: Если refresh_token истечёт, потребуется повторить процесс авторизации с шага 1.
